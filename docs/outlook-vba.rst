Outlook VBA Scripts
===================

This VBA Module provides a feature to write a selection of emails to disk, such that the .msg file is named
with date_sent -- sender -- subject and the visible attachments are saved alongside the file with the same date_sent as a prefix.

This documentation needs a lot of work to be more useful!
